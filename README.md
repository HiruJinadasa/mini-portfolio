# Mini Portfolio

This repository contains beginner-level projects to demonstrate my learning in Python, SQL, and QA Automation.

## Project 1: Sales Data Analysis (Python)
- File: `sales_analysis.py`
- Tools: Python  
- Description: Creates sample sales data, calculates total sales, and finds the region with the highest sales.

## Project 2: Login Automation Test (Conceptual – Selenium)
- File: `login_test.py`
- Tools: Python, Selenium  
- Description: A conceptual Selenium script showing how a login page can be automated.
- Note: This script is for understanding automation flow and is not executed yet.

## Project 3: Sample SQL Queries
- File: `sample_sql_queries.sql`
- Tools: SQL  
- Description: Creates a sample employees table and demonstrates basic SQL queries such as filtering and aggregation.

## Project 4: AI Student Performance Prediction (Conceptual – Machine Learning)  
- Tools: Python, Pandas, Scikit-learn  
- Description: A sample machine learning script that demonstrates how a model can be trained to predict student performance based on study hours.  
- Note: This is a conceptual example to show understanding of the basic AI/ML workflow including data preparation, model training, and prediction.

## Project 5: Data Science – Online Store Customer Analysis  
- Tools: Python, Pandas, Matplotlib, Seaborn  
- Description: Analyzed a sample dataset of online store customers to explore purchase behavior, average spending by gender, and visit frequency.  
- Note: Beginner-friendly project designed to show understanding of the data science workflow including data exploration, analysis, and visualization.
